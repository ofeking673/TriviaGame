Documentation of Trivia Server:

Login:
	Request:
		RequestId = 0
	
		client sends: 
			string username
			string password
	Response:
		
		If succeful:
			status = 100
			new handler = menu request handler
		if failed:
			status = 108
			new handler = login request handler

Signup:
	Request:
		RequestId = 1

			string password
			string email
	Response:
		
		If succeful:
			status = 110
			new handler = menu request handler
		if failed:
			status = 118
